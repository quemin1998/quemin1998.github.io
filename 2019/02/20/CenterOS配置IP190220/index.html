CenterOS 7 配置静态IP

  1. 使用命令查看当前系统使用IP：ifconfig

  2. 若当前使用的账号不是root账号，可能无法修改ip配置文件，所以需要使用命令切换当前账号为root账号(你得晓得root密码是啥)

    切换root用户命令(登陆超级管理员)：su

    输入密码 ( 输入时不显示输入内容，但确实输入了 )

  3. 进入/etc/sysconfig/network-scripts/目录

    切换目录：cd

  4. 查看当前目录所有文件:ls

  1. 修改ifcfg-eth0文件(需要超级管理员权限才能保存)

    修改某文件：vim ifcfg-eth0

    进入修改文件界面后，按 i 编辑文件

    使用主机cmd，查看当前网络：arp /a

    使用IP：192.168.29.86(局域网IP)

    Redis预留：192.168.29.87，192.168.29.88

    按ESC，输入 :wq 保存退出

  2. 最后就是最有意义，最有价值，体验成果的时候

    使用 service network restart 重启网络服务

    或者使用 reboot 直接重启服务器

    然后：ping www.baidu.com

    要是ping不通，那就完犊子了，重新去检查哪儿出问题了吧

    ifconfig 查看IP是否变更，重启服务没用就重启服务器，一遍不成再走一遍，再去查查对应文档挽救一下

上述内容为正常的CenterOS7系统，ifconfig查看时显示的为eth0网卡信息的正常配置，若你得系统是eth0网卡的话，那么配置起来问题不大，一下就配的出来
但是作为一个菜鸟,通过VMware虚拟机简易安装的CenterOS,没有发现什么eth0配置文件,根本没地儿改,是另一个ifcfg-eno16777736的文件,还好配置文件一般都是ifcfg打头,应该有些小伙伴是一样得情况,那是网卡网络配置温江,而且在安装时没有去特意设定CenterOS网络的,安装之后默认没有开启网络配置
所以我在配置IP时，修改ifcfg-eno16777736文件

完成后使用 service network restart 重启网络服务，就完事儿了
